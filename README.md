# Apple-Social-Media-Analytics for Strategic Branding
-> Social Media Analytics Case Study using Excel
## Project Overview
This project analyzes Apple's social media performance across multiple platforms to derive strategic branding insights. The dashboard was built using Microsoft Excel.
## Tools used
1.Microsoft Excel
2.Pivot Tables
3.Power Query
4.Charts & Visualizations
## Project Files

## Key Analysis Performed
->Platform-wise engagement analysis
->Follower growth trends
->Content performance comparison
->Brand sentiment indicators
